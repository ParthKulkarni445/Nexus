# Student Placement & Internship Platform – Project Planning
## Goal of the Platform

Create a **central platform for students** to:

* Track placement timelines
* Access company information
* Read real student experiences
* Analyze placement statistics
* Prepare better for interviews

## 1. Student Side

### Profile

* Student profile creation
* Basic details
* Resume upload
* Academic details
* Skills & interests

### Timeline

Displays placement process stages:

* PPT (Pre-Placement Talk)
* OA (Online Assessment)
* Interview

Each stage includes:

* Date
* Time
* Status updates

Example:

```
PPT → OA → Interview
Date & Time shown for each stage
```

---

# 2. Company Section

### Company Registration

Companies can register and provide:

* Company name
* Roles offered
* Internship / Placement tag
* Eligibility criteria
* Selection process stages

### Recruitment Process

Typical flow:

1. Registration
2. PPT
3. Online Assessment
4. Interviews

---

# 3. Blog / Experience Section

Students can share their placement or internship experiences.

### Blog Features

* Title
* Description
* Formatting support:

  * Bold
  * Italic
  * Emojis
  * Rich text
* Images
* PDF upload
* External links
* Download option

### Tags & Filtering

Blogs can be filtered by:

* Company tag
* Internship / Placement
* Year
* Role
* On-campus / Off-campus

### Bookmark

Users can bookmark useful blogs.

### Moderation

* Blog submission request
* Admin approval
* Report option for inappropriate content
* Admin tool for removal

### Comment Section

* Discussion area under blogs
* Comment threads
* Community interaction

---

# 4. Company Statistics

Each company page shows:

### Basic Information

* Company Name
* Roles offered

### Compensation

* CTC (for placements)
* Stipend (for internships)

### Placement Stats

* Number of students placed
* Placement percentage

### Eligibility

* Eligible branches
* CGPA criteria

### Data Views

Statistics displayed as:

* Year-wise
* Company-wise
* Graphs / charts
